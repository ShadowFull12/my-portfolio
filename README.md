## ⚡ Overview

This repository contains the source code for my interactive portfolio. It was designed to go beyond standard static pages by implementing strict `scrollYProgress` physics, pure CSS math-based animations, and 3D space transformations to create an immersive, cinematic breakdown of my journey, achievements, and technical stack. 

### 🚀 Stack
- **Framework:** [Next.js](https://nextjs.org/) (App Router format)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animation:** [Framer Motion](https://www.framer.com/motion/) + Native CSS Physics
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** [Vercel](https://vercel.com/) (Ready)

---

## 💻 Local Development

1. **Clone the repository:**
```bash
git clone https://github.com/ShadowFull12/aritra-portfolio.git
cd aritra-portfolio
```

2. **Install dependencies:**
```bash
npm install
```

3. **Run the local development server:**
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## 🌐 Deployment
This project is structurally optimized for 1-click deployment to **Vercel**. 
Simply link the repository to your Vercel account, and the `build` scripts natively handle static optimization.
